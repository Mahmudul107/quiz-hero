1. What is the differences between Map, filter, forEach and find in JavaScript?

↓

3. What is Event loop and how does it works ?

↓

Ans: An Event Loop is a way for programs to handle multiple tasks or events at the same time. It works by continuously checking for events and running the appropriate code for each event when it happens. This allows programs to be more efficient and responsive to user actions.

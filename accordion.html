1. What is the differences between Map, filter, forEach and find in JavaScript?

↓

Ans: map. The 'map' method is used to repeat over an array or collection and transform each element into a new value, the filter method filtered out elements that do not meet a certain condition, the forEach method perform a certain operation on each element, the find method return the first element that meets a certain condition, based on a function you provide.
